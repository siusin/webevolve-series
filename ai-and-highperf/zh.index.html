活动简介

Web 进化论 · 高性能 Web 应用的新探索

在日常开发中，使用现有的 Web 技术直接实现一个高效的跨平台应用常常面临性能和交互体验方面的诸多挑战。尽管 Web 平台具有强大的普及性和兼容性，但传统的 Web 技术在处理复杂的用户界面和跨平台需求时，往往在性能、响应速度以及流畅性上不能满足移动应用的高标准。

此次技术研讨会旨在汇聚行业专家，共同探讨如何定义、实现和测试一套符合现有 Web 技术规范的 Web 技术子集，专门优化用于移动端场景，注重 UI 和用户体验优先，强调程序运行与 UI 渲染的独立性，避免传统的 UI 渲染和程序执行相互阻塞的问题。通过重新设计与简化现有的部分 Web 技术接口，能够使其更适合移动端设备的高效渲染和操作体验。此外，引入符合移动端特性的交互效果，也将提升整体应用的流畅度与用户体验，推动 Web 平台在移动场景中的高性能表现。

通过这次研讨会，我们期望建立一个高性能跨平台 Web 应用子集社区组，与行业专家共同探索如何通过标准化和创新，推动 Web 技术在跨平台应用中的持续进化。

Web 进化论 · 共建 WebAI 未来

随着人工智能（AI）技术的飞速发展，Web 平台逐渐成为实现 AI 应用和服务的重要载体。WebAI 凭借其隐私友好的特点、低延时、低成本、广泛的开发者生态和免安装可离线的优势，吸引了越来越多的开发者和用户的关注。尽管 WebAI 具备诸多优势，但仍面临一些技术和标准化方面的挑战，特别是在隐私保护、安全性、跨平台兼容性和AI模型的透明性等问题上。为了解决这些问题并推动 WebAI 的健康发展，W3C 希望能够在早期阶段组织相关的技术讨论和标准制定，促进 Web 与 AI 的融合发展。

此次技术研讨会将为开发者、研究人员和行业专家提供一个交流的平台，深入探讨 WebAI 的关键技术挑战，并探讨如何通过标准化机制确保 WebAI 技术在性能、安全性、隐私保护等方面的最佳实践。同时，本次活动也将重点分享 WebAI 在提高Web应用开发生产效率方面的可行用例，以及如何共同探讨可标准化的 API，为开发者提供更高效的工具和方法。
