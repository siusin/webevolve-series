讲者简介

François Daoust (W3C)

François Daoust 是 W3C（万维网联盟）的资深媒体技术标准专家，深耕 Web 和媒体的技术融合，领导 W3C 媒体产业技术发展，同时也在 W3C 技术战略团队中负责媒体相关国际技术标准的演进战略。François 现在担任着 W3C Web 媒体工作组、第二屏幕工作组、Web GPU 工作组、Web 游戏社区组的团队负责人。François 也曾参与孵化与引领 Mobile Web 最佳实践、Web 与 TV、WebRTC 等领域的国际标准化，为相关行业发展作出了巨大贡献。

联系讲者: email fd[at]w3.org | Github

顾扬（英特尔）

顾扬，英特尔公司 Web 图形和 Web AI 负责人。本科毕业于浙江大学竺可桢学院，硕士毕业于浙江大学计算机科学与技术学院。2004年加入英特尔，2011年带领团队开始 Web 图形相关研发，包括 WebGL 1.0优化，WebGL 2.0标准及浏览器实现，WebGPU 标准及浏览器实现，TensorFlow.js和ONNX Runtime WebGPU 后端开发等，累计贡献超过4300个补丁。羽毛球爱好者，曾任英特尔上海羽毛球俱乐部主席7年。

联系讲者：email yang.gu[at]intel.com

胡宁馨（英特尔）

胡宁馨，Intel 首席工程师，服务于 Web 平台工程团队，专注 Web 技术多年，现为 W3C 机器学习工作组 Web Neural Network API (WebNN) 规范的发起者和联合编辑，Chromium WebNN 模块代码联合负责人。

联系讲者：email ningxin.hu[at]intel.com ｜ GitHub

高纯（声网）

高纯，声网高级架构师，主要负责 Web 引擎、Web 媒体处理、媒体实时传输等技术的研发工作。自2019年起担任声网在 W3C 的 AC 代表。在2010年至2018，曾在 Intel 开源技术中心工作，先后参与 MeeGo、Tizen、Android N Auto、ChromeOS、AliOS 等开源操作系统项目，以及 WebKit、Chromium、Crosswalk 等 Web 引擎项目的研发。

徐乾伟（蚂蚁集团）

徐乾伟，蚂蚁 Galacean 3D 互动引擎团队负责人。2015年开始在阿里从事移动端 WebGL 应用开发，2019年加入蚂蚁，带领团队进行 Galacean 3D 引擎的研发，2021年在 Github 开源引擎。熟悉 3D 引擎技术和互联网公司的图形技术应用实践，热爱Web和图形学，崇尚开源精神。

赵磊（中国移动咪咕）

赵磊，中国移动咪咕前端开发总监，毕业于吉林大学计算机科学与技术学院，专注 Web 开发10余年，在咪咕主要负责视频彩铃、数字人、云渲染等技术的研发工作，目前在 W3C 重点参与小程序工作组、媒体娱乐兴趣组以及 Web 与网络兴趣组等相关领域的标准化工作。

联系讲者：email zhaolei[at]migu.cn ｜ GitHub
